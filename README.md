# PD_Anastasija_Salti-uka
## Projekta nosaukums: Ābolu ievārījuma kalkulators
## Apraksts: Programma aprēķina nepieciešama cukura daudzumu atkarībā no ābolu svara, kopējās izmaksas ievērojot cukura cenu un gala ievārījumu masu pēc vārīšanas laika, kur pazūd 10% no kopejas masas.

## Funkcionalitāte:
- [] Ievads 
- [] cuk_daudz 
- [] kop_iz
- [] gal_masa
- [] Izvads

## Palaišana:
1 Iejupieladet python.
2 Iejupieladet abolu_ievarijuma_kalkulators.py
3 Kommandrindā ievadiet python abolu_ievarijuma_kalkulators.py
